REWARDS FOLDER
==============

This folder holds the files that play when a reward tier is unlocked.

GOLD REWARD (unlocked at 400 points)
--------------------------------------
Drop a video file named exactly:

    reward-gold.mp4

...into this folder. The app will play it automatically when the
Grand Master reward is unlocked.

Any MP4 video file works. Recommended: a short music video or
favourite song (1–3 minutes). Keep the file size reasonable for
a tablet — under 200 MB is fine.

FUTURE REWARDS
--------------
Silver and Bronze tiers currently show a badge only.
To add a video to those tiers, edit config.js and set the
videoFile property for the relevant tier.
